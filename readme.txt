== Description ==

Tumblrize is a small and convenient Wordpress plugin to post to Tumblr whenever you hit the “publish” button. It uses Tumblr’s simple API.

It’s lightweight, it just requires you to input your tumblr email and password in the plugin editor screen.

After that, you’re ready to cross-post!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

   1. Download the zip file: Tumblrize (v 1.0)
   2. Unzip the file in your /wp-content/plugins/ directory
   3. Go to your wordpress administration panel and activate the plugin
   4. Go to Plugins > Editor and choose Tumblrize in the dropdown menu
   5. Edit $tumblr_email    = “”; and $tumblr_password = “”; with your login credentials
   6. That’s it!
